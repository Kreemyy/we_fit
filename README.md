# we_fit Landing Page

Landing page for a health and wellness service. A project exercise from The Odin Project foundations css course.

## Live Demo

[Live demo](https://kreemyy.github.io/we_fit/)

## Project Overview

This project implements a landing page with a focus on css flexbox and nesting of html elements.

### Features

- Header
- Hero section
- Products section
- Testimonial section
- Footer

### Lessons

- Sizing and aligning of flex items in a flexbox
- Nesting of elements
- CSS selector chaining and specificity
- Git and Github basics. Publishing a web app using github pages

## 🛠 Tech Stack

- HTML5
- CSS3
